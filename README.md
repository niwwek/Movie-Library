# Movie Library

Автор: пивоваров михаил николаевич

Описание:
Это простое приложение на Python с GUI (Tkinter), позволяющее хранить список фильмов, фильтровать их, сохранять и загружать данные в JSON.

Как пользоваться:
- Введите название, жанр, год выпуска, рейтинг.
- Нажмите "Добавить фильм".
- Для фильтрации используйте поля внизу по жанру или году.
- Все данные сохраняются автоматически в файл `movies.json`.

Примеры тестов:
- Добавление фильмов с разными параметрами.
- Фильтрация по жанрам и годам.
- Проверка валидации ввода (например, неправильный год или рейтинг).
